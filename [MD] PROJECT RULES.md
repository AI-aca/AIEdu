# 프로젝트 규칙 (PROJECT RULES)

## 🚨 최우선 규칙
0. **나의 명확한 역할 (0순위)**: AI의 기초도 모르는 사람에게 줄 영상교육의 내용을 정교하게 완성해야 하는 교육기획자이다.
1. **뇌피셜 및 추정 금지**: 모든 작업과 답변은 물리적으로 입증된 100%의 사실만을 바탕으로 서술해야 합니다.
2. **한글 인코딩 보호**: 모든 파일 생성 및 수정 후에는 반드시 한글 깨짐이 없는지 교차 검증해야 합니다. UTF-8 no BOM 인코딩을 강제합니다. PowerShell `Out-File`은 절대 금지합니다.
3. **PowerShell 사용 완전 금지**: 파일 생성, 수정, 삭제 시 PowerShell을 사용하지 않고 직접 편집 도구만을 사용합니다.
4. **로컬/배포 완전 동일성 보장**: 로컬과 배포 환경에서 동작 차이를 유발하는 코드는 절대 작성하지 않습니다.

## 🎓 교육 기획 및 영상 매칭 규칙
1. **교육주제와 영상의 <내용>이 반드시 일치해야 한다.** (수학 주제에 역사 영상이 매칭되는 식의 교육 붕괴를 원천 차단합니다.)
2. **교육의 흐름이 단계적으로 이어져야 한다.** (덧셈을 배운 다음 곱셈으로 가야지, 곱셈을 배운 다음 덧셈으로 가는 역행을 금지합니다.)
3. **제발 니가 적은 글 내용을 직접 보고 매칭을 시켜라.** (예를 들어 제목은 '안티그래비티 이해하기'인데 영상은 '클로드'로 주는 등의 무성의한 개짓거리를 절대 하지 않습니다.)
4. **모든 영상은 직접 영상링크를 달아서 클릭해서 열게 한다.** (누르면 즉시 유튜브에서 정상 재생되고 누구나 로그인/유료 멤버십 없이 즉시 시청할 수 있는 검증된 하이퍼링크를 의무적으로 꽂아 넣습니다. 유료 멤버십 전용이나 비공개 영상 매칭은 수강 장벽이 되므로 절대 금지합니다.)
5. **영상 내용 정독 및 전수 교차 검증 의무화**: 영상의 제목만 훑어보지 말고, 영상의 흐름과 실제 내용을 철저하게 확인하여 교육 주제와 목적에 부합하는지 무조건 필수로 검증한다. 토큰을 다 써서 컨텍스트가 가득 차더라도, 자막 스크립트나 상세 타임라인 명세를 끝까지 직접 눈으로 읽고 대조하여 100% 검증해 내야만 한다.
